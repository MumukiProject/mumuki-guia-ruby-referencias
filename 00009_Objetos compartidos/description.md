¿_Te acordás de Fito_? `Fito` también tiene un amigo, `Juli`. `Juli` es nieto de `AbueloGervasio`. Cuando `Juli` es feliz `Fito` es feliz: 

```ruby
module Fito
  def self.amigo=(un_amigo)
    @amigo = un_amigo
  end
  
  def self.es_feliz_como_su_amigo?
    @amigo.felicidad > 105
  end
end
```
